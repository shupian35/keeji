# 视频播放功能 + 按导入类型条件显示

## 目标
1. 在笔记详情页实现真正的视频播放功能
2. 视频导入时显示播放器，文本导入时隐藏播放器
3. 隐藏播放器时笔记内容占满宽度

## 实施步骤

### Step 1: 添加 `video_player` 依赖
**文件**: `pubspec.yaml`
```yaml
video_player: ^2.9.3
```
运行 `flutter pub get`

### Step 2: 模型添加 `sourceType` 字段
**文件**: `lib/models/video_record.dart`
- 新增 `SourceType { video, text }` 枚举
- `VideoRecord` 添加 `sourceType` 字段，默认 `SourceType.video`（向后兼容）
- 更新 `copyWith` 方法

### Step 3: 代码生成
```bash
dart run build_runner build --delete-conflicting-outputs
```

### Step 4: 数据库迁移
**文件**: `lib/database/app_database.dart`
- 版本号 1 → 2
- 添加 `onUpgrade`：`ALTER TABLE video_records ADD COLUMN source_type INTEGER NOT NULL DEFAULT 0`
- 更新 `_videoFromMap` / `_videoToMap` 读写 `source_type`

### Step 5: 导入页面设置 sourceType
**文件**: `lib/features/import/import_page.dart`
- 视频导入：`sourceType: SourceType.video`
- 文本导入：`sourceType: SourceType.text`

### Step 6: 重写视频播放器
**文件**: `lib/features/viewer/widgets/video_player_widget.dart`
- 使用 `video_player` 包实现真正的播放
- 功能：播放/暂停、进度条拖拽、音量开关、时间显示
- 错误状态 UI

### Step 7: ViewerPage 条件布局
**文件**: `lib/features/viewer/viewer_page.dart`
- 用 `if (video.sourceType == SourceType.video)` 包裹视频播放器 + 分隔线
- 隐藏时 Markdown 内容自动占满宽度

### Step 8: 添加 l10n 字符串（可选）
**文件**: `lib/l10n/app_zh.arb`, `app_en.arb`, `app_zh_TW.arb`
- 添加 `"videoPlaybackError": "无法播放此视频"` 等

### Step 9: 验证
```bash
flutter analyze
flutter run -d windows
```
- 导入视频 → 笔记详情页显示播放器，可播放
- 导入文本 → 笔记详情页无播放器，笔记占满宽度
- 旧数据自动兼容（默认 sourceType=video）
